## Filing Details

| Item | Detail |
|---|---|
| **Filing system** | [TEAS Plus](https://www.uspto.gov/trademarks/apply) ($250/class) or TEAS Standard ($350/class) |
| **Total cost (2 classes)** | $500 (TEAS Plus) or $700 (TEAS Standard) |
| **TEAS Plus requirements** | Must select description from USPTO ID Manual, email correspondence, pay all fees upfront |
| **Filing basis** | Intent to Use (1b) — cheaper to start, file Statement of Use later when product launches |
| **Timeline** | ~8-12 months to registration if no opposition |

### TEAS Plus vs TEAS Standard

- **TEAS Plus ($250/class):** Cheaper but you must pick your goods/services description from the [USPTO ID Manual](https://idm-tmng.uspto.gov/id-master-list-public.html). The descriptions above are written to align with existing ID Manual entries.
- **TEAS Standard ($350/class):** More flexibility to write custom descriptions. Use this if the USPTO rejects the TEAS Plus descriptions.

**Recommendation:** Try TEAS Plus first to save $200.

## Timeline & Process

1. **File application** — Submit via [TEAS](https://www.uspto.gov/trademarks/apply) (~30 min)
2. **Examining attorney review** — USPTO assigns an attorney (~3-4 months)
3. **Office action (if any)** — May request clarifications, you have 3 months to respond
4. **Publication for opposition** — Mark published in Official Gazette for 30 days
5. **Notice of Allowance** — If no one opposes (Intent to Use filing)
6. **Statement of Use** — You have 6 months (extendable to 36 months) to show the mark in commerce
7. **Registration** — Trademark registered on the Principal Register
